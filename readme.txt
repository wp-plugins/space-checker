== Description ==
Plugin to check free and used space on the server. Plugin can send mail to administrator and activate read-only mode
 - If you want to know when your server use more than 80% (for example) and send email for you.
 - If you want to activate read-only mode (disable add\\edit\\remove posts, pages, comments)
 - If you want to activate maintenance mode

This plugin for you!


== Installation ==
1. Upload `space-checker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to `Space Checker` settings
