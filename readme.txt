Plugin to check free and used space on the server. Plugin can send mail to administrator and activate read-only mode

== Description ==
Plugin to check free and used space on the server, prevent blog and database corruption and safe you data.
= What can do this plugin? =
1. Plugin can send mail to administrator
2. Can activate read-only mode (disable add\\edit\\remove posts, pages, comments)
3. Can show to user message about maintenance on the public site
4. Show the plugin status, free and used space via widget on Admin Dashboard

= Who need this plugin? =
1. Who wants to save their data on the blog
2. Who wants to know the amount of free and used space on the site
3. Who wants to activate the read-only mode

== Installation ==
1. Upload `space-checker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to `Space Checker` settings
